About app

This project is part of my journey at the incredible programming school "Alem".

Groupie-Tracker is a web application that helps people get to know cool groups/artists more closely. It has taught me to work with APIs and manipulate data to create a site that displays information.

The website uses an API to store data about artists/groups, with each entry consisting of four fields:

  1. Artists: contains information about bands and artists, such as their name(s), image, start year, date of their first album, and members.
  2. Locations: shows the last and/or upcoming concert locations.
  3. Dates: displays the last and/or upcoming concert dates.
  4. Dates: displays the last and/or upcoming concert dates.

Additionally, the website features a search function that provides suggestions as users type in a query. Users can also filter the results based on the creation date, first album date, locations of concerts, and number of members.

With these added features, Groupie-Tracker offers an even more robust way for people to explore and discover their favorite bands and artists.
